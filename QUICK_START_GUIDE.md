# HarmonyOS智能步数平台 - 快速启动指南

## 🚀 项目简介
这是一个基于HarmonyOS的运动健康应用，提供步数记录、积分兑换、排行榜等功能。

## 📋 系统要求
- HarmonyOS SDK 6.0.2(22) 或更高版本
- DevEco Studio 最新版本
- 支持HarmonyOS的设备或模拟器

## 🔧 快速开始

### 1. 项目导入
1. 打开DevEco Studio
2. 选择 "Open" 或 "Import Project"
3. 选择项目根目录
4. 等待项目同步完成

### 2. 环境配置
项目已预配置好以下环境：
- 网络权限
- 存储权限
- 传感器权限（预留）

### 3. 运行项目
1. 连接设备或启动模拟器
2. 点击 "Run" 按钮或使用快捷键
3. 等待应用安装和启动

## ⚙️ 配置说明

### AppConfig.ets 配置项
```typescript
// API基础配置
baseUrl: 'https://api.example.com'  // 后端API地址
enableMock: true                    // Mock数据开关
timeout: 10000                     // 请求超时时间
enableLog: true                    // 日志开关
```

### 切换到真实后端
1. 修改 `AppConfig.ets` 中的 `baseUrl`
2. 设置 `enableMock: false`
3. 配置真实的后端服务

## 🐛 常见问题

### Q: 应用启动失败
**A**: 检查设备连接和权限配置，确保HarmonyOS版本兼容。

### Q: 网络请求失败
**A**: Mock模式下无需网络，真实模式下检查网络连接和API地址。

### Q: 步数数据不更新
**A**: 当前为模拟数据，需要集成真实传感器功能。

### Q: 页面跳转异常
**A**: 检查路由配置，确保所有页面都在 `main_pages.json` 中注册。
